# Installation

1. Install the requirements: `python3 -m pip install -r requirements.txt`
2. Create .env file with OpenAI API Key:
    - `cp env_EXAMPLE .env`
    - `vim .env`
5.  Run the tool: `python3 hermeneisGPT.py --help`
